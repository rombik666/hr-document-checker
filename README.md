# HR Document Checker

HR Document Checker — учебный прототип системы проверки HR- и бизнес-документов с использованием rule-based проверок, RAG-подсистемы и семантических агентов.

Система принимает документы в формате DOCX/PDF, извлекает текст и структуру, определяет тип документа, выделяет ключевые сущности, выполняет формальные и семантические проверки, после чего формирует итоговый отчёт с приоритетами Critical / Major / Minor.

## Основные возможности

- загрузка DOCX/PDF-документов;
- парсинг текста, абзацев, таблиц и PDF-страниц;
- классификация типа документа: резюме, сопроводительное письмо, анкета кандидата, вакансия;
- классификация секций: contacts, experience, skills, education, projects, summary;
- извлечение сущностей: e-mail, телефон, ссылки, даты, навыки;
- формальные проверки:
  - полнота документа;
  - наличие контактов;
  - наличие опыта, навыков и образования;
  - проверка структуры;
  - проверка дат в опыте;
- семантические проверки:
  - слабые формулировки;
  - шаблонные фразы;
  - потенциальные противоречия;
  - релевантность резюме требованиям вакансии;
- RAG-подсистема для поиска контекста в базе знаний;
- итоговый отчёт с группировкой Critical / Major / Minor;
- блок релевантности вакансии;
- сохранение отчётов в БД;
- режим no_store для проверки без сохранения отчёта;
- маскирование e-mail и телефонов при сохранении отчёта;
- минимальный веб-интерфейс;
- Swagger API;
- Docker Compose стенд с PostgreSQL;
- логирование запросов и базовые метрики;
- автоматические тесты.
