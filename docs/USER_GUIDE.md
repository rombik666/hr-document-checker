# User guide

## Назначение системы

HR Document Checker помогает выполнить первичную проверку HR-документа:

- резюме;
- сопроводительного письма;
- анкеты кандидата;
- описания вакансии.

Система не принимает кадровые решения. Она формирует рекомендации и технические замечания.
