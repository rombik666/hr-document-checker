# Final project summary

HR Document Checker — прототип системы проверки HR- и бизнес-документов с использованием ИИ-агентов.

## Реализовано

- импорт DOCX/PDF;
- извлечение сущностей;
- классификация документов и секций;
- формальные агенты;
- семантические агенты;
- RAG и vector search;
- provider-agnostic LLM interface;
- итоговый отчёт;
- БД;
- безопасность/no_store;
- Web UI;
- Swagger;
- DOCX export;
- Docker/PostgreSQL;
- Prometheus/Grafana;
- backup/restore;
- тесты.

## Статус

Прототип готов к демонстрации и соответствует основным требованиям ТЗ и итогового отчёта.
