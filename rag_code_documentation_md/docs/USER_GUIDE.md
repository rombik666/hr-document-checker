# User guide

## Как пользоваться

1. Открыть `http://127.0.0.1:8000/web/`.
2. Загрузить DOCX/PDF.
3. При необходимости вставить вакансию.
4. Выбрать режим хранения: `temporary`, `metadata_only` или `no_store`.
5. Нажать «Запустить проверку».
6. Посмотреть отчёт.
7. Если отчёт сохранён, скачать DOCX.

## Приоритеты

- Critical — критичная проблема.
- Major — значимое замечание.
- Minor — небольшое замечание.

## Экспорт отчёта

Если отчёт сохранён, доступна ссылка «Скачать DOCX».

API:

```text
GET /api/v1/documents/reports/{report_id}/export/docx
```

## Важно

Система не принимает кадровые решения, а только формирует рекомендации.
